# Explora Tours El salvador
## Es una página informativa y moderna diseñada para que la agencia "Explora Tours El Salvador" pueda mostrar sus paquetes de viaje en un solo lugar, permitiendo a los clientes ver los destinos disponibles y enviar sus datos fácilmente para ser contactados.
Es una sola página web donde el usuario puede entrar, ver las fotos y precios de los viajes disponibles en El Salvador, y llenar un formulario sencillo para pedir información sin salir de la misma pantalla.
